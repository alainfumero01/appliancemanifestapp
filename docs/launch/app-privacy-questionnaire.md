# LoadScan App Privacy Draft

## Data Collected

### Contact Info
- Email Address
Purpose:
- Account creation and authentication
- Support communication

### User Content
- Appliance sticker photos
- Manifest titles, load references, line items, exported manifest data
Purpose:
- Core app functionality
- Cloud sync and export

### Identifiers
- User ID
- Organization ID
- Subscription product ID
Purpose:
- Account management
- Entitlement enforcement
- Team access control

## Data Not Used for Tracking
- LoadScan does not use collected data for third-party advertising or cross-app tracking.

## Data Linked to the User
- Email address
- Account identifiers
- User content
- Subscription status / organization membership

## Sensitive Data Notes
- Sticker images may include product labels and serial/model information entered by the user.
- Payment data is handled by Apple through App Store subscriptions, not by LoadScan directly.

## Required Final Review Before Submission
- Confirm whether exported files or logs contain additional metadata
- Confirm whether crash reporting or analytics SDKs are added before release
- Re-answer App Privacy form if analytics, support chat, or third-party logging are introduced
